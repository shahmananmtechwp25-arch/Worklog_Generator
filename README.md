# ImpactLog AI: M.Tech Project
**Problem:** Professionals struggle to quantify daily work for annual reviews.
**Solution:** A Python-based tool that uses LLMs to convert raw notes into STAR-method achievements.

### Core Features:
- **Global Access:** Deployable via Streamlit Cloud.
- **Offline Mode:** Uses SQLite for local data persistence.
- **AI Transformation:** Uses GPT-4o-mini for professional rewriting.

### How to Run:
1. `pip install -r requirements.txt`
2. `streamlit run Project.py`
